# SSH Server Agent - All-in-One

Telegram бот + Web интерфейс для мониторинга серверов

## Функции

### Telegram Bot
- ✅ Добавление серверов
- ✅ Мониторинг метрик (CPU, RAM, Disk)
- ✅ Выполнение команд
- ✅ Автоматические алерты

### Web Interface
- ✅ Dashboard с метриками
- ✅ Терминал (выполнение команд)
- ✅ Файловый менеджер
  - Просмотр файлов и папок
  - Редактирование текстовых файлов
  - Загрузка файлов
  - Скачивание файлов
  - Создание папок
  - Удаление файлов
